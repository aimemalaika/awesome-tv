##### Getting Started
- clone the repository by running\
    `git clone https://github.com/aimemalaika/awesome-tv.git`
- navigate to the folder\
    `cd awesome-tv`
- Install packages\
    `npm install`
- Run the app\
    `npm start`
- Create build app\
    `npm run build`
- Test Linters app\
    `npm run check-linters`
